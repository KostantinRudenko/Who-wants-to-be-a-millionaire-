1. git clone ссылка на продукт - скачиваем на компьютер продукт с GitHub
2. git checkout -b название_ветки - создаем новую ветку и переключаемся на нее 
3. git branch - проверить на какой ты ветке
4. git status - проверить состояние проекта
5. git add . - добавить ВСЕ файлы в коммит
6. git commit -m "сообщение здесь" - фиксирует все изменения в ветке и оставляет о них комментарий
7. git push - отправить изменения на GitHub
8. git pull - скачать изменения с GitHub (НЕ ПУТАТЬ С GIT CLONE)
9. git switch название_ветки - переключаемся на существующую ветку
